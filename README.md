## Description

My-Top is a simplified re-implementation of the Unix `top` command written in C.
The project focuses on monitoring system processes in real time by retrieving and parsing system information from the `/proc` filesystem.

## Project Goals

- Understand process monitoring in Unix systems
- Work with the `/proc` filesystem
- Parse system data efficiently
- Manage dynamic terminal display
- Improve low-level systems programming skills

## Technologies Used

- C language
- Unix system calls
- `/proc` filesystem
- Process management
- Makefile

## Implemented Features

- Reading process information from `/proc`
- Displaying active processes
- CPU usage monitoring
- Memory usage retrieval
- Dynamic screen refresh
- Error handling

## Compilation

make

## Usage

- ./my_top
- press 'q' to quit
